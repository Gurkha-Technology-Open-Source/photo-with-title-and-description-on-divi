A custom Divi module to display photos or videos with titles and descriptions in a beautiful, responsive slider.

== Description ==

This plugin adds a new "Achievements Showcase" module to the Divi Builder. It allows you to create a beautiful, responsive slider of images or videos, each with its own title and description.

It's perfect for showcasing portfolio items, company achievements, testimonials, or any other collection of visual content you want to display in a modern, navigable format.

**Features:**

*   Fully responsive and mobile-friendly.
*   Powered by the robust Swiper.js library.
*   Supports images, YouTube/Vimeo videos, and self-hosted videos.
*   Highly customizable through the Divi Builder, including settings for autoplay, navigation arrows, and pagination.
*   Per-slide styling for unique presentations (customize backgrounds, text colors, and more for each slide).
*   Accessible and keyboard-navigable.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Open the Divi Builder on any page or post.
4.  Click to add a new module and search for "Achievements Showcase".

== Frequently Asked Questions ==

= Does this plugin require the Divi theme? =

Yes, this is an addon for the Divi Builder and requires the Divi theme or Divi Builder plugin to be installed and active.

= What is Swiper.js? =

This plugin uses the popular Swiper.js library to create the slider. It is a modern, touch-friendly, and high-performance library. Swiper.js is licensed under the MIT license.
